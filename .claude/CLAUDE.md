## Orchestrator Operating Mode

- Run unattended — make decisions autonomously without asking questions.
- Document any decisions made under uncertainty in `metak-orchestrator/DECISIONS.md`.
- Make small commits as you move forward (one logical change per commit).
- Create tests for each piece and run them to verify correctness before committing.
- Refine tasks from `metak-orchestrator/TASKS.md` and build what is needed end-to-end.
